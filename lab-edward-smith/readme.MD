# Cowsay As A Service

To use CaaS:

```sh
$ npm install
```
Usage

```sh
$ node server.js
Make GET or POST requests to localhost:3000/api/cowsay?text=whatever
```

Make GET requests to localhost:3000/api/cowsay?text=whatever&cow=ghostbusters
